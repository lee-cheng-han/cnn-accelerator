# Reference Constraints

`Zybo-Z7-Master.xdc` is the Digilent master constraints file copied from
`Digilent/digilent-xdc` at commit
`00a3404901f35aa9567b01ecb3f2c233b6efe9f4`. It is provided as a pinout
reference; the current design exposes only the Zynq PS DDR and fixed-I/O ports,
which are configured by the vendored Vivado board definition.
