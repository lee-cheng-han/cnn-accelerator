# v1.0.0-preboard

This pre-release freezes the board-arrival baseline for the Arty Z7-20 image-to-image CNN accelerator.

## Verified Baseline

- Full RTL, golden tensor, AXI, and system-wrapper regression passing.
- Routed at 125 MHz with WNS `+0.133 ns` and WHS `+0.016 ns`.
- Zero failing setup or hold endpoints.
- Vivado warning budget: zero unknown warnings, critical warnings, or errors.
- Bitstream, XSA, bare-metal ELF, FSBL, and `BOOT.BIN` rebuilt from the tagged source.

## Attached Evidence

- `system_wrapper.bit`: programmable-logic bitstream.
- `arty_z7_20_cnn.xsa`: exported hardware platform including the bitstream.
- `cnn_baremetal.elf`: image-to-image DMA golden-test application.
- `BOOT.BIN`: SD-card boot image containing the FSBL, bitstream, and application.
- `flow_report.md`: timing, utilization, and artifact summary.
- `v1.0.0-preboard.sha256`: SHA-256 checksums for the four deployable artifacts.

Physical UART, throughput, setup-photo, and optional ILA evidence remain pending board arrival.
